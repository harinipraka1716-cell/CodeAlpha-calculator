# CodeAlpha Advanced Calculator

## Project Overview
This is an Advanced Calculator developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Sin, Cos, Tan
- Logarithm
- Dark/Light Mode
- Keyboard Support
- Calculation History
- Live Digital Clock
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the project in VS Code.
3. Open index.html using Live Server.
4. Start using the calculator.
